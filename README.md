# 杭州萧山新千西人工智能应用软件有限公司 · 企业官网

独立企业官网项目，仅包含公司展示页面，不含多元宇宙叙事平台或其他业务模块。

## 页面

- `/` — 首页（关于我们、核心业务、企业优势、联系我们）
- `/privacy` — 隐私政策
- `/terms` — 服务条款

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm start
```

## 部署

推荐部署至 [Vercel](https://vercel.com)，绑定域名 `www.xinqianxi.world`。
